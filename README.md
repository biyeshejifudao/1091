# Java Web服装购物网站

## 项目简介
服装购物网站是基于Java Web，Springboot框架，Mysql数据库的服装在线购物网站。角色上分为：1.管理员角色  2.普通用户角色， 功能上包含前台系统：负责展示，推荐，加入购物车，下单  后台系统负责：负责服装管理、用户管理、订单管理等。适合作为毕业设计、大作业、课程设计等。
